# 太陽系ダイナミクス・シミュレーター

このプロジェクトは、TVD Triadic Velocity Dynamics理論に基づき、太陽系のダイナミクス（力学）を可視化・シミュレートするWebアプリケーションです。

## 概要
- 惑星や太陽の運動をシミュレーションし、軌道や速度の変化を視覚的に表示します。
- TVD理論の教育・研究用途や、物理学への興味喚起を目的としています。

## ファイル構成
- `main.html` : シミュレーター本体のHTMLファイル

## 使い方
1. `main.html` をブラウザで開いてください。
2. シミュレーションが自動的に開始されます。
3. 惑星の動きや速度ベクトルなどを観察できます。

## 必要環境
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
