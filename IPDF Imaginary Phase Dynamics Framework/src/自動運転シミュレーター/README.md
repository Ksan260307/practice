# 自動運転シミュレーター

このプロジェクトは、IPDF（Imaginary Phase Dynamics Framework）理論に基づいた自動運転の挙動をシミュレートするWebアプリケーションです。

## 概要
- 仮想空間上で自動運転車両の動作や経路計画、障害物回避などを可視化します。
- IPDF理論の教育・研究用途や、自動運転技術への理解促進を目的としています。

## ファイル構成
- `main.jsx` : シミュレーター本体のReactコンポーネント

## 使い方
1. 必要に応じてReact環境でビルドしてください。
2. `main.jsx` をReactプロジェクトに組み込むか、ビルド後のアプリをブラウザで開いてください。
3. シミュレーション画面で自動運転の挙動を観察できます。

## 必要環境
- Node.js および npm
- React 17以降
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
