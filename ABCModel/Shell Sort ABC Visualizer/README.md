# Shell Sort ABC Visualizer

シェルソートのアルゴリズムを「生感(A)」「評価(B)」「俯瞰(C)」の心理力学モデルで解釈・可視化するインタラクティブ学習ツール。

## 概要

このアプリケーションは、Shell Sort（シェルソート）アルゴリズムの実行過程を **ABC Model** という感情・認知モデルの観点から分析し、リアルタイムで可視化します。単なるソートアルゴリズムの可視化にとどまらず、アルゴリズムの内部状態が心理力学的にどのように変化するかを追跡できます。

## 主な機能

### 1. **リアルタイム配列可視化**
- 20要素の配列を棒グラフで表示
- 処理中の要素をハイライト表示
- Gap戦略による分割状態を視覚化

### 2. **ABC Model メトリクス**

#### A: Authenticity（生感・アクション）
- 要素の移動・交換に伴うエネルギー
- 配列の実体的再構成を意味します
- 高い値 = 動的な変化が発生

#### B: Aesthetic（評価・基準）  
- ソート順序を守るべき基準への圧力
- 比較が行われるたびに上昇
- 値のズレが見つかると修正エネルギーを生成

#### C: Meta（俯瞰・戦略）
- 「どの程度の細かさで見るか」の戦略
- Gap値に依存：大きいGap = 大局的、小さいGap = 局所的
- メタ認知的な処理レベルを表現

### 3. **動的グラフ分析**
- A、B値と RuinScore の時系列推移を表示
- アルゴリズムの心理力学的な振る舞いを追跡

### 4. **操作コントロール**
- **START/PAUSE** ：自動実行の開始・一時停止
- **ステップ実行** ：1ステップ進める
- **リセット** ：初期状態に戻す
- **速度調整** ：アニメーション速度を変更

## 使い方

### 起動
```
npm install  # 依存パッケージをインストール
npm start    # 開発サーバーを起動
```

### 基本操作
1. **START ボタン**をクリックするとシェルソートが自動実行されます
2. **ステップ ボタン**で1段階ずつ進めながら細部を観察できます
3. **SPEED スライダー**で実行速度を調整します
4. **リセット ボタン**で最初からやり直します

### 観察ポイント
- Gapが大きい時と小さい時でABC値がどう変わるか
- 要素の交換直後にvAとvBがどう反応するか
- RuinScore（混乱度）の最大値と回復パターン
- 最終的に STABILITY REACHED 状態になるまでの過程

## ABC Model について

ABC Model は CORE 3.2.0 に基づいた心理力学フレームワークです：

| モデル | 意味 | シェルソートでの役割 |
|--------|------|---------|
| **A** | Authenticity（生感） | 要素の実体的な動き・交換の中身 |
| **B** | Aesthetic（評価） | 理想（昇順）との比較圧・ズレの検出 |
| **C** | Meta（俯瞰） | 処理の粒度・戦略の切り替え(Gap値) |

**RuinScore** は A、B、C の組み合わせメトリクスで、全体的な「混乱度」を示します。

## カスタマイズ

### 配列サイズを変更する
`main.jsx` の以下の行を編集：
```javascript
const ARRAY_SIZE = 20;  // ここを変更（例：30、50など）
```

### ABC計算ロジックを調整する
`updateABC()` 関数内のパラメータをカスタマイズすることで、シェルソート各段階でのABC値の振る舞いを変更できます。
