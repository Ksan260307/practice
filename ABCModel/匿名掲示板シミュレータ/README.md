# 匿名掲示板シミュレータ

ABCモデルと Gemini AI を組み合わせた、**匿名掲示板の集団心理をシミュレート** するツール。スレッド内の力学パラメータを調整することで、AIが生成する返信のトーン（攻撃的・冷笑的・標準的）がどのように変化するかを観察できます。

## 概要

匿名掲示板（2ちゃんねるなど）の最大の特徴は、**個人の責任感がなくなること**によって発生する、独特の集団心理です。

このシミュレーターは：
- Firebase でスレッドと投稿を永続的に管理
- Gemini API でAIが投稿内容をリアルタイム生成
- ABCモデルのパラメータで、スレッド内の雰囲気・心理状態を制御
- 匿名掲示板特有の文化（名無しさん、sage、安価など）を実装

という方法で、掲示板の集団心理動学を可視化します。

## 主要機能

### 匿名掲示板風UI

- **昭和のデザイン**: 懐かしの2ちゃんねるを模した古風なUI
- **スレッド一覧**: 更新順に表示（最新がトップ）
- **投稿ビュー**: 掲示板の伝統的な番号・名前・本文・時刻フォーマット
- **軽量かつシンプル**: 複雑な機能は排除

### ABCモデル統合エンジン（ThreadDynamicsEngine）

スレッド内の心理力学を **4つのパラメータ** でシミュレート：

#### vA: スレッド内の行動力・活発性
- **意味**: スレッドの投稿活動、エネルギー
- **範囲**: 0-2
- **AIの反応**: 高いほどエネルギッシュな返信

#### vB: 攻撃性・プレッシャー
- **意味**: 議論の激しさ、非難の度合い
- **範囲**: 0-2
- **AIの反応**: 高いほど攻撃的な返信が増加

#### vC: 冷笑・俯瞰性
- **意味**: ユーザーの冷笑的態度、高みから見下ろす視点
- **範囲**: 0-2
- **AIの反応**: 高いほど冷笑的・ニヒリスティックな返信

#### vE3: 外部環境ストレス
- **意味**: スレッド外部の社会情勢・ニュース・ストレス
- **範囲**: 0-2
- **AIの反応**: vB （攻撃性）を上昇させる

### AIの自動投稿

- **周期**: 12～20秒のランダム間隔で自動生成
- **容量**: 50～150文字程度の簡潔な投稿
- **文体**: ThreadDynamicsEngine が計算した tone に基づく
- **要素**: 安価（>>[番号]）、「ｗ」、「…」などの掲示板的な表現

### トーン制御

計算されたパラメータに基づいて、AIの返信スタイルが自動変更：

| 条件 | トーン | 例 |
|------|--------|-----|
| vB > 1.2 | **攻撃的** | 「お前が悪い。無能の極み。」 |
| vC > 1.2 | **冷笑的** | 「はいはい、また同じこと言ってるよｗ」 |
| その他 | **標準的** | 「確かにそうだね。俺も同じ経験ある。」 |

### ユーザー操作機能

- **新规スレッド作成**: タイトルと初期投稿を指定
- **投稿作成**: 任意の名前（またはsage）で投稿
- **ページスクロール**: 新しい投稿が追加されると自動スクロール
- **AI自動投稿 ON/OFF**: 「設定」で自動投稿のオン/オフを切り替え

## 使用方法

### 基本ワークフロー

1. **アプリが起動**
   - Firebase認証が初期化される（匿名認証）
   - 既存スレッド一覧が読み込まれる

2. **スレッドを作成/選択**
   - 「新規スレッド」で新しいスレッドを作成
   - 既存スレッドをクリックして閲覧開始

3. **投稿する**
   - 「投稿」フォーム に内容を入力
   - 任意の名前を指定（空欄なら「名無しさん」）
   - 送信ボタンをクリック

4. **AIの自動返信を観察**
   - 投稿から12～20秒後、AIが自動返信
   - AIのトーンはパラメータに基づいて動的に変化

5. **パラメータ調整**
   - 「設定」タブで vA、vB、vC、vE3 を調整
   - 変更と同時に、AIのスタイルが動的に変更される

### インタラクティブな実験

#### 実験1: 標準的な掲示板
```
vA: 1.0,  vB: 0.5,  vC: 0.5,  vE3: 0.0

結果: AIが平穏で建設的な返信をする
メッセージ例: 「確かにそうだね。俺も同じ経験ある。」
```

#### 実験2: 荒れたスレッド（攻撃的）
```
vA: 1.5,  vB: 1.8,  vC: 0.5,  vE3: 0.0

結果: AIが攻撃的で非難的な返信をする
メッセージ例: 「お前が悪い。無能の極み。」
トーン自動切り替え: 標準 → **攻撃的**
```

#### 実験3: 冷笑的スレッド（ニヒリズム）
```
vA: 1.2,  vB: 0.6,  vC: 1.9,  vE3: 0.0

結果: AIが高みから見下ろす冷笑的態度
メッセージ例: 「はいはい、また同じこと言ってるよｗ」
トーン自動切り替え: 標準 → **冷笑的**
```

#### 実験4: 社情報ストレス増加
```
vA: 1.0,  vB: 0.8,  vC: 0.8,  vE3: 1.5

結果: vE3がvBを上昇させる（vB: 0.8 → 1.1）
スレッドが徐々に攻撃的に変化
```

#### 実験5: 過度な監視（C暴走）
```
vA: 1.5,  vB: 0.5,  vC: 2.0,  vE3: 0.0

計算:
curA = 1.5 - 2.0 × 0.4 = 0.7 (クランプ: 0.7)
curB = 0.5 + 0.0 × 0.3 = 0.5
curC = 2.0 + 0.5 × 0.2 = 2.1 (クランプ: 2.0)
ruinScore = 2.0 × 2 = 4.0（危険）

結果: スレッド内の雰囲気が毒性を帯びる
```

## 実装詳細

### ThreadDynamicsEngine

```javascript
class ThreadDynamicsEngine {
  calculate() {
    const clamp = (value) => Math.max(0, Math.min(2, value));
    const curA = clamp(vA - vC * 0.4);  // 冷笑性が行動力を減衰
    const curB = clamp(vB + vE3 * 0.3); // 環境ストレスが攻撃性を上昇
    const curC = clamp(vC + vB * 0.2);  // 攻撃性が冷笑性を強化
    const ruinScore = Math.min(6, Math.max(curA, curB, curC) * 2);
    return { curA, curB, curC, ruinScore };
  }
}
```

**計算ロジック**:
1. **Cが高い** → **Aが減衰**: 過度な冷笑が行動力を奪う
2. **E3が高い** → **B上昇**: 外部ストレスが攻撃性をあおる
3. **B高い** → **C上昇**: 攻撃的な雰囲気が冷笑を強化

### Gemini API 呼び出し

```javascript
const systemPrompt = `
あなたは日本の匿名掲示板の住人です。
【ルール】
1. 本文のみ。ヘッダーや「名前：」は不要。
```

**AIの振る舞い**:

## 匿名掲示板文化の実装

### 名前機能
- 空欄または「sage」で「名無しさん」に変換
- 任意の名前を指定可能
- sage 機能で更新日時を変えない（本実装では簡略化）

### ユーザーID
- `uid` と `threadId` をハッシュ化して短縮ID生成
- スレッドごとに異なるIDが割り当てられる（実装例：`A1B2C3D4`）

### 安価（引用）
- AIが稀に「>>[番号]」という形式で前の投稿を引用
- 掲示板の伝統的な返信形式

### タイムスタンプ
- 年/月/日(曜日) HH:MM:SS.mm 形式を実装
- ミリ秒単位の精密度で、投稿の時系列を完璧に追跡

1. vB を上げて「攻撃的なスレッド」を作成
2. vC を上げて「冷笑的なスレッド」を作成
3. 各パラメータでのAIの返信スタイルの違いを実感

### 上級: システムの相互作用を理解
1. ThreadDynamicsEngine の計算ロジックを理解
2. C→A (冷笑が行動力を減衰) の効果を検証
3. vE3 が vB に影響する、社会ストレスの連鎖を観察
