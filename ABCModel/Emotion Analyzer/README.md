# Emotion Analyzer

テキストから深層心理を可視化する感情分析ツール。ABCモデルと感情体系 v1.0に基づいて、心理状態の危険度を評価します。

## 概要

Emotion Analyzerは、ユーザーが入力したテキストを Google Gemini APIで分析し、以下の情報を抽出・可視化します：

- **感情分析**: 9つの基本感情の強度
- **詳細感情**: Layer-2の具体的な感情語
- **ABCモデル**: 心理的パターン（A: 行動力、B: こだわり、C: 客観性）
- **RuinScore**: 心理的破綻リスク（0-6スケール）

## 主要機能

### 感情体系 Layer-1（基本感情）

9つの基本感情を0.0～1.0のスケールで評価します：

| 感情 | ラベル | 説明 |
|------|--------|------|
| joy | 喜び・楽しさ | ポジティブで充実した状態 |
| expectation | 期待・ワクワク | 未来への期待感 |
| trust | 安心・信頼 | 安定感と信頼感 |
| surprise | 驚き | 予想外の出来事への反応 |
| sadness | 悲しみ・孤独 | 喪失感や寂しさ |
| fear | 恐れ・不安 | 恐怖や不安感 |
| anger | 怒り・苛立ち | 怒りや不満 |
| disgust | 嫌悪 | 嫌悪感 |
| contempt | 軽蔑 | 軽蔑や見下し |

### 感情体系 Layer-2（詳細感情）

テキストから3～5個の具体的な感情語を抽出します。

### ABCモデル分析

各モジュールについて以下の3つのパラメータを評価：

#### A: Authenticity（行動力・モチベーション）
- **v（強度）**: Low(0) / Mid(1) / High(2)
- **Δ（変動）**: Stable(0) / Swing(1) / Spike(2)
- **状態**: Normal / Runaway / Zero

#### B: Aesthetic（こだわり・責任感）
- **v（強度）**: Low(0) / Mid(1) / High(2)
- **Δ（変動）**: Stable(0) / Swing(1) / Spike(2)
- **状態**: Normal / Runaway / Zero

#### C: Meta（客観性・気配り）
- **v（強度）**: Low(0) / Mid(1) / High(2)
- **Δ（変動）**: Stable(0) / Swing(1) / Spike(2)
- **状態**: Normal / Runaway / Zero

### RuinScore（危険度スコア）

心理的破綻リスクを0～6で評価：

| スコア | レベル | 説明 | アクション |
|--------|--------|------|-----------|
| 0-2 | 安定（R0） | 心理的破綻リスク極低 | 現状維持 |
| 3-4 | 注意（R1） | ストレスや感情偏り有 | セルフケア推奨 |
| 5 | 危険（R2） | 激しい変動や暴走状態 | セルフケア必須 |
| 6 | 破綻/極限（R3） | システム限界状態 | 即座の休息・外部支援要 |

## 技術スタック

- **フレームワーク**: React 18+
- **スタイリング**: Tailwind CSS
- **UI コンポーネント**: lucide-react
- **AI API**: Google Generative AI (Gemini 1.5-flash)
- **レスポンス形式**: JSON

## セットアップ

### 必要な環境

- React 18 以上
- Google AIプラットフォームAPIキー

### インストール

```bash
npm install
npm start
```

### 環境変数

APIキーの設定方法：

```javascript
// コード内で直接設定（本番環境では推奨されません）
const apiKey = "YOUR_API_KEY_HERE";

// または環境変数経由：
const apiKey = process.env.REACT_APP_GEMINI_API_KEY;
```

## 使用方法

1. **テキスト入力**: テキストエリアに今感じていることや最近の出来事を自由に入力
2. **解析実行**: 「解析を開始する」ボタンをクリック
3. **結果確認**: 以下の情報が表示されます：
   - 感情バランスグラフ（Layer-1）
   - ABCモデルの状態カード
   - RuinScoreと危険度評価
   - AI分析サマリー
   - 詳細感情（Layer-2）

## UIコンポーネント

### EmotionBar
感情の強度をプログレスバーで表示。1%未満は非表示。

### ABCCard
各モジュール（A、B、C）の詳細情報をカード形式で表示。

### RuinScoreDisplay
危険度スコアをアイコン付きで大きく表示。色分けで直感的に識別可能。

## エラーハンドリング

- **APIキーエラー**: Googleアカウント設定の確認が必要
- **空の応答**: 入力テキストが長すぎないか確認
- **ネットワークエラー**: 自動重試機能（最大5回）搭載

## パフォーマンス最適化

- 感情バーは1%未満は非表示（UI整理）
- 遅延ローディング対応
- アニメーション（duration-1000 ease-out）で結果表示

## レスポンシブデザイン

- モバイル対応（4px～8pxのパディング柔軟調整）
- タッチフレンドリーなボタンサイズ
- 横幅制限（max-w-5xl）でデスクトップでも最適表示

## 注記

- テキスト入力は暗号化されません（プライバシー上の配慮が必要な場合は注意）
- APIコール数に応じた使用料が発生する可能性があります
- Gemini APIの利用制限に準じます
