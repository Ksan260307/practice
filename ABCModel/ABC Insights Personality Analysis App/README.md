# ABC Insights Personality Analysis App

ABCモデルに基づいた個人の性格特性・行動パターン分析ツール。チームメンバーの心理的特性を可視化し、組織内のコミュニケーション最適化に活用します。

## 概要

ABC Insightsは、3つの心理的パラメータと4つの環境ストレス要因を用いて、個人の性格特性と現在の心理状態を分析するプロフェッショナル診断ツール。

## 主要機能

### ABCモデル（3つの性格特性）

#### A: Authenticity（行動力・モチベーション）
- **専門家表記**: 体感/自然感
- **機能**: 行動力・モチベーション・自発性
- **特徴**: やる気・エネルギー・主導性の度合い

#### B: Aesthetic（こだわり・責任感）
- **専門家表記**: 基準/評価感
- **機能**: こだわり・責任感・完璧主義
- **特徴**: 価値観・理想像・品質基準へのこだわり度

#### C: Meta（客観性・気配り）
- **専門家表記**: 俯瞰/監視認知
- **機能**: 客観性・気配り・全体把握
- **特徴**: 客観的思考・周囲との調和・配慮性

### 環境ストレス要因（EF: Environmental Factors）

| ラベル | 専門家表記 | 説明 |
|--------|----------|------|
| E0 | Health | 睡眠・体調の悪化 |
| E1 | Environment | 職場環境のストレス |
| E2 | Social | 対人・評価のプレッシャー |
| E3 | Noise | タスク過多・雑音 |

### タイプ診断

#### Cタイプサブタイプ
- **C1**: 構造俯瞰型（全体像・仕組みから物事を捉える）
- **C2**: 場面整序型（場の空気や進行、整頓を重視）
- **C3**: 監視-自己意識型（他者の目や見られ方を意識）
- **C4**: 社会文脈型（社会通念や常識・大義を重んじる）

#### Bタイプサブタイプ
- **B1**: 内向き基準（「自分の理想」を重視）
- **B2**: 外向き評価（「他者からの評価・承認」を重視）

### 分析タブ
- **Analysis**: メンバーの性格パラメータを視覚化
- **Comparison**: 複数メンバーの特性比較
- **Insights**: AI分析による詳細レポート通知

### 管理機能
- メンバー追加・削除
- 性格パラメータの手動調整
- 履歴保存とレポート生成
- 専門家モード/一般モード切り替え

## 技術スタック

- **フレームワーク**: React 18+
- **スタイリング**: Tailwind CSS
- **UI コンポーネント**: lucide-react
- **状態管理**: React Hooks (useState, useMemo, useEffect)
- **データ形式**: JSON

## 使用方法

### 基本操作
1. メンバー追加: 左サイドバーの「+」ボタンをクリック
2. パラメータ設定: 各ABCスライダーで強度と変動性を設定
3. 環境要因入力: E0-E3スライダーで外部ストレス要因を設定
4. サブタイプ選択: CタイプおよびBタイプを選択
5. 分析表示: 複数タブで多角的に分析

### UIモード
- **一般モード**: 分かりやすいラベルと説明
- **専門家モード**: 詳細な心理学用語と専門的な解釈

### レスポンシブデザイン
- デスクトップ: 左サイドバー + メインコンテンツ
- モバイル: ハンバーガーメニュー搭載（自動開閉）

## パラメータ構造

```javascript
{
  id: '1',
  name: '名前',
  role: '役職',
  notes: 'メモ',
  params: {
    vA: 0-2,        // A特性の強度
    deltaA: 0-2,    // A特性の変動性
    vB: 0-2,        // B特性の強度
    deltaB: 0-2,    // B特性の変動性
    vC: 0-2,        // C特性の強度
    deltaC: 0-2,    // C特性の変動性
    E0: 0-2,        // 体調ストレス
    E1: 0-2,        // 環境ストレス
    E2: 0-2,        // 対人ストレス
    E3: 0-2,        // 雑音ストレス
  },
  cType: 'C1' | 'C2' | 'C3' | 'C4' | 'None',
  bType: 'B1' | 'B2' | 'None'
}
```

## カラースキーム

- **A（Authenticity）**: ローズレッド（bg-rose-50, text-rose-500）
- **B（Aesthetic）**: アンバー（bg-amber-50, text-amber-500）
- **C（Meta）**: ブルー（bg-blue-50, text-blue-500）

## ラベル定義

### 強度（Intensity）
- 0: 低
- 1: 中
- 2: 高

### 変動性（Delta）
- 0: 安定
- 1: やや揺らぐ
- 2: 不安定

### 環境要因（EF）
- 0: なし
- 1: 少し
- 2: 強い

## 機能詳細

### Analysis タブ
- 現在選択されたメンバーの詳細プロフィール表示
- ABCパラメータとEF要因の可視化
- サブタイプ情報の表示

### Comparison タブ
- 複数メンバーのABC特性を並列表示
- チームの特性バランスを把握

### Insights タブ
- AI分析による心理診断レポート
- ストレス警告メッセージ
- 推奨される対応策の提示

## デフォルトメンバー

初回起動時に以下のサンプルメンバーが読み込まれます：

```
名前: 田中 太郎
役職: シニアエンジニア
特性: Aが高め（主導性強）、Bが高め（こだわり強）、Cは中程度
Cタイプ: C2（場面整序型）
Bタイプ: B1（内向き基準）
```
