# 画風チューナー（Art Style Tuner）

## 概要
**画風チューナー**は、ABCモデルに基づき、4つのパラメータを調整することで、あらゆる画風を細かくコントロールできるAI画像生成アプリケーションです。

自分の想像する絵柄を直感的に操作でき、Google Imagen APIを使用した高品質な画像生成を実現します。

## 主な機能

### 🎨 ABCモデルベースの画風制御
アプリケーションは ABCModel v3.2.0 に基づいており、以下の4つのパラメータで画風を操作します：

| パラメータ | 役割 | 説明 |
|-----------|------|------|
| **筆の勢い（Passion）** | 生感・表現力 | 低：繊細、高：力強い筆致 |
| **ゆらぎ（Chaos）** | 偶然性・テクスチャ | 低：均一、高：不規則でグランジ |
| **写実性（Realism）** | 現実性 | 低：抽象的、高：写真的リアリズム |
| **構図（Design）** | 配置・構成 | 低：スナップショット、高：グラフィックな構成 |

### 📝 カスタムプロンプト入力
描きたいモチーフを自由に入力し、画風パラメータと組み合わせて画像生成を行います。

### 🔄 複合ルール
パラメータ間の相互作用により、より豊かな表現が可能です：
- **筆の勢い＋ゆらぎ高＆写実性低** → 表現主義・アヴァンギャルド
- **写実性高＆構図高** → 高級コマーシャルアート・コンセプトアート

## 使い方

1. **スライダーを調整** - 左のパネルから4つのパラメータをそれぞれ調整
2. **プロンプト入力** - 「描きたいもの」をテキストボックスに入力
3. **生成ボタンをクリック** - 「画像を生成」ボタンをクリック
4. **結果確認** - 生成された画像と使用されたプロンプトを確認

## 技術仕様

- **フレームワーク**: React（Hooks）
- **API**: Google Generative AI API（Imagen 4.0）
- **UIコンポーネント**: lucide-react（アイコン）
- **リトライ機能**: 指数バックオフ対応（最大5回）

### パラメータマッピング
ユーザーUIの 0-100 スケールは、内部の 0.0-2.0 スケールに変換されます：
```
Core Value = (UI Value / 100) × 2.0
```

## セットアップ

API Keyが Canvas環境で自動注入される想定です。

## 画風キーワード生成ロジック

各パラメータの値に応じて、以下のように画風キーワードが自動生成されます：

### Authenticity（筆の勢い）
- **0-0.5**: delicate lines, flat colors
- **0.5-1.5**: expressive strokes, natural painting style
- **1.5-2.0**: thick impasto, raw artistic energy

### Aesthetic（写実性）
- **0-0.5**: highly stylized, abstract, surreal
- **0.5-1.5**: stylized but anatomically sound
- **1.5-2.0**: hyper-realistic, photorealistic

### Meta（構図）
- **0-0.5**: snapshot-like, unorganized
- **0.5-1.5**: well-framed, balanced
- **1.5-2.0**: poster-like, minimalist

## カスタマイズ例

### 和風なイラスト例
- **筆の勢い**: 70（表現力重視）
- **ゆらぎ**: 30（ある程度均一）
- **写実性**: 40（やや様式化）
- **構図**: 60（バランス取れた配置）
- **プロンプト**: "A beautiful woman in Japanese kimono"

### 現代アート例
- **筆の勢い**: 80以上（力強い表現）
- **ゆらぎ**: 70以上（グランジ要素）
- **写実性**: 20（抽象的）
- **構図**: 70（グラフィカル）
