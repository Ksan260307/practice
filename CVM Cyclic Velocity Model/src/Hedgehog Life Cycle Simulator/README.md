# Hedgehog Life Cycle Simulator

This project is a web application that simulates the life cycle of a hedgehog based on the CVM (Cyclic Velocity Model) theory.

## Overview
- Visualizes and simulates the various stages of a hedgehog's life using cyclic velocity dynamics.
- Designed for educational and research purposes, as well as for intuitive understanding of biological and physical phenomena.

## File Structure
- `main.jsx`: Main React component of the simulator

## Usage
1. Build the project in a React environment if necessary.
2. Integrate `main.jsx` into your React project or open the built app in your browser.
3. Observe the simulation of the hedgehog's life cycle on the screen.

## Requirements
- Node.js and npm
- React 17 or later
- Modern web browser (Google Chrome, Firefox, Edge, etc.)
