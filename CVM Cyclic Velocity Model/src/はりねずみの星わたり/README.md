# はりねずみの星わたり

このプロジェクトは、CVM（Cyclic Velocity Model）理論を応用した「はりねずみの星わたり」シミュレーターです。

## 概要
- はりねずみが星の上を渡る様子を、周期的な速度モデル（CVM）に基づいてシミュレート・可視化します。
- CVM理論の教育・研究用途や、物理現象の直感的理解を目的としています。

## ファイル構成
- `main.html` : シミュレーター本体のHTMLファイル

## 使い方
1. `main.html` をブラウザで開いてください。
2. シミュレーションが自動的に開始されます。
3. はりねずみの動きや軌跡を観察できます。

## 必要環境
- モダンなWebブラウザ（Google Chrome, Firefox, Edgeなど）
