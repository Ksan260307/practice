# VIF STRIKER2

このプロジェクトは「VIF STRIKER2」と題したアプリケーションです。

## 概要
- VIF（Velocity-Impulse Framework）の考え方を応用したストライカー系シミュレーションまたはゲームの第2弾です。
- 主な実装ファイル: `main.html`, `VIF.html`

## 使い方
1. `main.html` または `VIF.html` をブラウザで開いてください。
2. シミュレーションまたはゲームを体験できます。

## ディレクトリ構成
```
OTHER/
└── src/
    └── VIF STRIKER2/
        ├── main.html
        ├── VIF.html
        └── README.md
```
