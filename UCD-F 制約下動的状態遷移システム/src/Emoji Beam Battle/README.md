# 📝 Emoji Beam Battle - README

**Emoji Beam Battle** は、**UCD-F_ABC（制約下動的状態遷移システム）**の設計思想に基づき開発された、ブラウザ上で動作する超軽量・高密度なサバイバル・バトルロイヤルゲームです。

## 🌟 1. ゲーム概要
プレイヤーは絵文字のエージェントとなり、広大なフィールドで99人のCPUと生き残りをかけて戦います。CPU同士も互いに争い、アイテムを奪い合う、常に状況が変化する戦場を生き抜いてください。

## 🚀 2. 主要機能
- **多彩な特殊武器**: ホーミング、貫通、広範囲爆発、一撃必殺など、アイテムによって戦略が激変。
- **進化するAI**: 敵同士の戦闘、間合いの管理（引き撃ち）、カニ歩き移動、障害物を回避する「パニックモード」を搭載。
- **動的パワーシステム**: 時間経過で連射力が低下するため、常に⚡を集めるか戦い続ける必要があります。
- **地形・残骸の活用**: フィールドの建造物だけでなく、倒れた敵の「死骸」もビームを防ぐ壁（盾）として利用可能。

## 🎮 3. 操作方法
このゲームはPCとスマートフォンの両方に最適化されています。

### 【PC操作】
- **移動**: `W` `A` `S` `D` または `矢印キー`
- **攻撃**: `マウスクリック`（長押しで自動連射）
- **エイム**: マウスカーソルの方向

### 【スマホ操作】
- **移動**: 画面左半分をスライド（仮想スティック）
- **エイム・攻撃**: 画面右半分をスライド（ツインスティック形式）

## 📦 4. アイテムリスト
- **💗 ハート**: HPを+1回復。本作唯一の回復手段。
- **⚡ サンダー**: 前方にランダムなジグザグ軌道を描く貫通ビーム。
- **🚀 ロケット**: 最寄りの敵を弱くホーミングする。
- **👻 ゴースト**: 気まぐれな軌道で飛び、一定時間後に敵を猛烈に追尾。
- **🎃 ボム**: 着弾地点で広範囲のダメージを与える。
- **👍 グッド**: 3秒後に巨大な👍を投下し、範囲内に3ダメージ。1発撃つと消滅。

## 🛠️ 5. 技術仕様 (UCD-F_ABC)
内部ロジックには、提供された設計書のエッセンスをエミュレートして組み込んでいます。
- **Bit-packed SoA**: エンティティの状態（HP、向き、武器等）を32bit整数にパックし、メモリ効率を最大化。
- **Spatial Hashing**: 空間ハッシュにより、100体のキャラと大量の弾幕の衝突判定を高速化。
- **Panic Mode Logic**: 壁にスタックしたことを検知すると、目的を一時破棄して逆方向・ランダム方向へ離脱。
- **Fixed-Point Emulation**: 浮動小数点の誤差を抑え、決定論的な挙動を担保。
