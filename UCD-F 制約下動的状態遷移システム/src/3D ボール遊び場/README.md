# 3D ボール遊び場

このプロジェクトは「3D ボール遊び場」と題したインタラクティブな3Dボールシミュレーションまたは遊び場アプリケーションです。

## 概要
- 3D空間内でボールを操作・観察できるシミュレーションまたは遊び場です。
- 主な実装ファイル: `main.html`

## 使い方
1. `main.html` をブラウザで開いてください。
2. 3Dボールの動きやインタラクションを体験できます。

## ディレクトリ構成
```
OTHER/
└── src/
    └── 3D ボール遊び場/
        ├── main.html
        └── README.md
```
